# SACGA-ESN
Self-Adaptive Cuckoo Genetic Algorithm (SACGA) for optimizing Echo State Networks (ESNs) for time series forecasting, focusing on the edge-of-chaos regime, Lyapunov-based dynamical stability, separation-driven representational capacity (Separation Ratio), and structured reservoir behavior.


# Paper

Giantsidi, S., & Tarantola, C. (2026).
*The self-adaptive Cuckoo genetic algorithm for echo state networks optimization.*
Digital Finance, 8(1), 16.


# Structure

* `src/optimization/` – SACGA implementation
* `src/models/` – ESN model
* `main.py` – run script


#  Usage

```bash
pip install -r requirements.txt
python main.py
```
